# SM.store - متجر إلكتروني

مرحباً بك في SM.store! موقع متجر إلكتروني حديث وسهل الاستخدام.

## 🎯 المميزات

✅ **إضافة المنتجات** - أضف منتجات جديدة بسهولة مع الصور والأوصاف والأسعار
✅ **عرض المنتجات** - واجهة جميلة وسهلة الاستخدام
✅ **تحميل الصور** - اختر صور من جهازك لكل منتج
✅ **معلومات المنتج** - أضف الوصف والسعر والحجم واللون والكمية
✅ **تكامل WhatsApp** - اطلب مباشرة عبر WhatsApp مع جميع البيانات
✅ **حفظ البيانات** - جميع المنتجات تُحفظ في متصفحك تلقائياً
✅ **تصميم متجاوب** - يعمل بشكل رائع على الهاتف والحاسوب

## 🚀 كيفية الاستخدام

### 1. فتح الموقع
- افتح ملف `index.html` في متصفحك
- أو اسحب الملف مباشرة إلى المتصفح

### 2. إضافة منتج جديد
1. اضغط على زر **"+ إضافة منتج"** في الأعلى
2. ملأ البيانات:
   - **اسم المنتج** - اسم واضح للمنتج
   - **الوصف** - وصف مفصل (مهم لجذب الزبائن!)
   - **السعر** - سعر المنتج
   - **اللون** (اختياري)
   - **الحجم** (اختياري)
   - **الكمية** - عدد القطع المتاحة
   - **صورة** - اختر صورة من جهازك
3. اضغط **"إضافة المنتج"**

### 3. عرض وإدارة المنتجات
- المنتجات تظهر في الصفحة الرئيسية مباشرة
- اضغط **"عرض التفاصيل"** لرؤية كل المعلومات
- اضغط **"حذف"** لحذف منتج

### 4. استقبال الطلبات
- عندما يضغط الزبون على **"اطلب عبر WhatsApp"**
- سيتم فتح WhatsApp مع رسالة تحتوي على:
  - اسم المنتج والسعر
  - الكمية المطلوبة
  - اللون والحجم
  - رابط للتواصل مباشرة: +212 728-958504

## 💾 حفظ البيانات
- جميع المنتجات تُحفظ تلقائياً في متصفحك
- لن تخسر البيانات حتى لو أغلقت الموقع
- لحذف جميع البيانات: امسح ذاكرة التخزين المؤقت للمتصفح

## 🎨 تخصيص الموقع
- لتغيير الألوان: عدّل المتغيرات في `style.css`:
  - `--primary-color` (الأخضر الرئيسي)
  - `--secondary-color` (الأخضر الداكن)
  - `--accent-color` (الأحمر)
